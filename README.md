# Airbnb Global Performance Dashboard

## Project Overview
This project analyzes Airbnb's global performance using data from various regions. The analysis is visualized in a Power BI dashboard, providing insights into revenue, booking trends, and property listings.

## Features
- **Global Reach**: Analysis across multiple cities and countries.
- **Interactive Visuals**: Power BI dashboard for deep diving into the data.
- **Data-Driven Insights**: Key performance indicators (KPIs) for hosts and guests.

## Data Source
The dataset includes:
- `Listings.csv`: Detailed information about property listings.
- `Reviews.csv`: Guest reviews and ratings.
- Data dictionaries for both datasets.

## How to View
To interact with the dashboard, open the `Airbnb Global Performance Dashboard.pbix` file using [Power BI Desktop](https://powerbi.microsoft.com/desktop/).

## Repository Structure
- `Airbnb Data/`: Raw CSV data files.
- `images/`: Extracted visuals from the project documentation.
- `Airbnb Global Performance Dashboard.pbix`: The Power BI project file.
- `Airbnb_Dashboard.pdf`: PDF report of the dashboard.
